Random words which holds some values...

i.e,
x = 15
y = 20

Variables x and y holds(stores) values location in memory.

Check memory locations of variables.

We can check with id()

i.e,
id( x )
id( y )

Rules to define variables

  • check


    Should be start with

    lower / upper case

    underscore ( _ )


    i.e,

    1. abc
    2. _xy
    3. PqR

  • clear


    Should not be start with

    digits ( 0 to 9)

    Should not be same as keywords name


    i.e,

    1. 156
    2. 42Name
    3. def
